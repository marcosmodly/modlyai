ModlyAI Widget - React Test

After building the widget, you'll need to use it in a proper React application.

To test the widget:

  1. Run npm run build in the widget directory
  2. Create a React app (using Create React App, Vite, or Next.js)
  3. Import the widget: import { FurnitureAIWidgetButton } from '@modlyai/furniture-widget'
  4. Use it in your component
