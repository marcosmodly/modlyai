ModlyAI Widget Button Test

This is a test page for the ModlyAI Widget Button. You should see a floating button in the bottom-right corner. Click it to open the widget in a modal popup.

Test Instructions:

  • Look for the floating button in the bottom-right corner
  • Click the button to open the widget modal
  • Try closing the modal by clicking the X button, clicking outside, or pressing ESC
  • The button color should adapt to the website's color scheme (currently using CSS variable --primary-color)

Color Customization:

The button automatically detects colors from CSS variables. Try changing the --primary-color variable in the style tag above to see it adapt.
